My personal website.

Powered by [Jekyll](https://jekyllrb.com) with [Minimal Mistakes Theme](https://mmistakes.github.io/minimal-mistakes/) fork by [Stuart Geiger](https://github.com/staeiou).  Inspired by [AcademicPages](https://github.com/academicpages/academicpages.github.io).

Makes use of [cv](https://github.com/CarlosP24/cv.git) to build CV.pdf.

Publications data is synced automatically with arXiv and Zotero. Other databases (talks, funding, outreach and awards) are built locally from tables in 'db'. 

For personalized use, build your own [cv](https://github.com/CarlosP24/cv.git), add your own .tsv dbs to 'db' and modify all relevant urls in '.github/workflows/pages.yml' (arXiv, Zotero, CV repo and name).
